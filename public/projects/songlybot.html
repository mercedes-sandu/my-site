Imaginarium

A Discord.py bot for the Overture Games Discord server.

July 2022 to August 2022 | Overture Games

SonglyBot is a small Discord API Python bot that I developed to help foster a community in the Overture Games Discord server. I had a really fun time learning to use Python and Discord API to create this bot, as well as Amazon Web Services to host it 24/7.

The bot has a variety of commands and useful features:

  • Assigning yourself roles by reacting with specific emojis.
  • Restricting access to the server until new members have read the rules.
  • Multiplayer Wordle (written in collaboration with a friend of mine).
  • A scoreboard for minigames.
  • Responding to certain keywords, for the sake of inside jokes.

I certainly plan on expanding the bot and adding more features as we release our first game and the community grows. If you have any ideas, I would love to discuss them!
